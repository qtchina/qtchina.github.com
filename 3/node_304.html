Adobe开放了RTMP协议规范
发布： 2009-06-16 21:31

Adobe又一个开源的东西。虽然开源，但商用还是要获得Adobe的许可。RTMP（the Real-time Messaging Protocol）协议作为客户端和服务器端的传输协议，这是一个专门为高效传输视频、音频和数据而设计的 TCP/IP 协议，使用 RTMP 协议传输的数据是未经加密的，包括用户名和密码等认证信息。
