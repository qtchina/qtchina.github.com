GNU Emacs 23.1 正式发布


2009-07-30 Toy Posted in NewsRSS



GNU Emacs 23.1 版本已经正式发布。该版本主要添加了 Anti-aliasing 字体支持、支持 D-Bus 和 Zeroconf、Emacs-daemon、支持 Xembed、支持 frame 透明、国际化增强（加入了 Chinese-GBK、Chinese-GB18030、Khmer 等新的语言环境）等特性。





参阅 News 文件了解进一步的信息。GNU Emacs 23.1 可从 GNU 的 FTP 站点下载。
