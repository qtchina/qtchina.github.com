稳定版内核kernel-2.6.22.2进入升级仓库
发布： 2007-08-10 14:55

Linus Torvalds 已经发布了 Linux Kernel 2.6.22 版。这是 Linux 内核的最新稳定版本，你可以从 The Linux Kernel Archives 网站下载到该版本。

Linux 内核 2.6.22 版的主要新增特性包括：




  • 使用 SLUB 分配器替换了 slab 分配器。

  • 添加了新的无线堆栈。

  • 增加了几个新的设备驱动程序。

  • 加强对IDE和SCSI硬件系统的支持,并增强了对 USB设备和3D加速卡的支持.
