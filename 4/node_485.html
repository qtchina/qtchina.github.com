I2P匿名网络研究
发布： 2010-06-20 16:51

I2P 是一个匿名网络项目，它提供了一个简单的网络层供对身份敏感的程序进行安全的匿名通讯。I2P 网络是动态的分布式网络，它在设计上并不信任网络中的任何一方，其中的所有数据都经过多层加密。

很多程序可以使用 I2P ，其用途涉及 Email，P2P，IRC 聊天等。

1. 安装
用source安装，
ant pkg
java -jar install.jar

用二进制包安装，下载后直接运行：
java -jar i2pinstall.exe
