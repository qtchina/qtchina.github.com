还没确定从哪版本开始支持的，我现在用的qt 4.7.1是已经支持了的，好像是在4.7.0版本加入的这个功能。 使用方法， CONFIG += link_pkgconfig # enable the PKGCONFIG feature PKGCONFIG += glib-2.0 # link against glib-2.0 另外，还可检没某个包是否存在， packagesExist(glib-2.0) { DEFINES += HAS_GLIB PKGCONFIG += glib-2.0 } // and in the code: #ifdef HAS_GLIB // use glib here #endif 这样使用pkgconfig的功能，已经非常简便了 还有一点疑问，这个在非linux系统上，如mac, win上是什么表现呢。
