概述

官方唯一的Qt4编程最佳实践指南

 

使用奇趣的(Trolltech) Qt 你可以编写运行在 Windows, Linux/Unix, Mac OS X 和嵌入式 Linux 系统下的工业级 C++ 应用程序，而不需要更改源代码。通过由奇趣权威专家所编写的这本书，你可以使用现有 Qt 最强大的版本 Qt4.1 取得最大的收获。

 

通过《Qt4 C++ 图形编程》，当你掌握了从 Qt 的模型/视图架构到 Qt 强大的2D绘图引擎，你会发现 Qt4 的最高效编程模式和技巧。书中作者首先向读者展示他对于 Qt 事件模型和布局系统无与伦比的洞察力。然后，通过实例介绍所有的高级编程技巧，从基本的 GUI 开发到高级的数据库与 XML 整合编程。

 

本书主要包括以下内容：

 

  • Qt4 的模型/视图架构和新的插件支持，以及 Qtopia 嵌入式编程的简单介绍。

  • 基本的对话框和窗口系统以及实现应用程序所需的所有基本概念。

  • 布局管理和事件处理的最佳实践。

  • 几乎所有的 Qt4 新的API , 包括强大的新 2D 绘图引擎和简单易用的容器类。

  • 每个章节都包含了完全的升级内容。

  • 其他书中没有包含的 Qt4 高级技巧，从创建 Qt 和应用程序的插件到使用本地API的接口。

  • 一个为 Java 开发者深入了解 C++/Qt 编程的附录。

 

配套 CD-ROM 中包含了 Windows, Mac, Linux, 类 Unix 及 minGW 平台上的 Qt4.1.1 开源版本，还有一整套的可自由获取的构建 Windows 应用程序的工具，以及本书示例的所有源代码。
