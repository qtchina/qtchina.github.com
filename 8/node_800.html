沉浸式学 Git

发布： 2014-04-20 14:50

沉浸式学 Git 是一份强调通过实践来掌握 Git 基础用法的指南。 本书包含 52 个实验，这些实验经过精心设计，篇幅皆十分短小， 只需几分钟时间便可完成。对于想要快速学习 Git 的朋友而言， 这是一本不可多得的好书。

本书原作者为 Ruby 编程语言大师 Jim Weirich，我利用业余时间 翻译完成。
