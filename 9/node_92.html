运用参考手册


Qt 的参考手册是每个 Qt 程序员必备的工具，因为它覆盖了 Qt 中所有的类和函数。这本书用到了 Qt 许多的类和函数，但不是它的全部，也不提供本书涉及到的类和函数的所有细节。要从 Qt 获得最大的利益，你应该尽快使自己熟悉 Qt 的参考手册。


这份手册可以从 Qt 的 doc/html 目录下以 HTML 的格式获取，并通过 web 浏览器读取。你还可以使用 Qt Assistant——Qt 的帮助浏览器，利用它强大的搜索和索引功能来比使用 web 浏览器更轻松和快速的获取帮助。要运行 Qt Assistant，Windows 上你可以通过点击开始菜单的 Qt by Trolltech v4.x.y|Assistant 按钮来启动 Qt Assistant，或者在 Unix 的命令行下键入 assistant，来启动它，或者在 Mac OS X 的 (finder) 中双击 Assistant。


手册首页"API Reference" 部分中的链接提供了浏览 Qt 类的不同方式。"All Classes" 页列出了 Qt API 中所有的类。"Main Classes" 页则只列出了最常用的 Qt 类。作为练习，你或许想要查阅我们在本章中用到的类和函数。
