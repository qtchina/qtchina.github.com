2.1. 安装 VLS

2.1.1. 在 Windows 上

在 VLS Windows download page,下载ZIP压缩文件，解压缩并运行里面的setup.exe.

2.1.2. 在 Linux 和 Mac OS X 上

2.1.2.1. 安装库文件

正常使用需要很多库文件:

  • libdvbpsi (always needed)

  • libdvdcss 如果你想访问加密的DVDs,

  • libdvdread 如果你想编码 DVDs成流,

  • libdvb 如果你想能从 DVB card (a satellite card or a digital terrestial TV card)编码流。

从VLS sources download page.下载库文件。

对每个库文件解压缩，配置(不包括libdvb 因为它没有./configure), 编译并安装:

% tar xvzf library.tar.gz
% cd library
% ./configure --enable-shared
% make
# make install

检查配置文件 /etc/ld.so.conf 是否有下面这行 :

/usr/local/lib

如果没有这行，加入，并运行ldconfig:

# ldconfig

2.1.2.2. 安装 VLS

下载最新的源程序包 : 从 VLS sources download page.获得vls-version.tar.gz文件，解压缩并生成./configure :

% tar xvzf vls-version.tar.gz
% cd vls-version
% autoconf

要锋利配置参数，打入 :

% ./configure --help

然后配置 vls :

  • 如果你要一个不带DVD支持的基本的VLS, 打入 : 

    % ./configure --disable-dvd

  • 如果你要带DVD支持的VLS,打入 :

    % ./configure

  • 如果你要带DVB支持的VLS，打入 :

    % ./configure --enable-dvb

然后，编译并安装 :

% make
# make install

如果必要，你可用运行make uninstall,make clean,或make distclean.

2.2. 删除 VLS

2.2.1. 在 Windows 上

打开"控制面板"，点击"添加删除程序"，选择 "VLS",点击"修改/删除".

2.2.2. 如果你从源程序编译VLS

转到包含VLS源程序的目录下并运行:

# make uninstall

然后删除VLS源程序。
